# Repo Explanation
- /SImpleServer is the node REST API backend
- /src you can find the react front end

# Running locally
- First, do not delete node modules. Just open the project from the base of the folder and run `yarn`
- Then `cd SImpleServer` and run `yarn` again
- Next you will have to import the mock data sent in csv format into a mongodb database called medical and into a table called patient
- Now you should be able to see run the app. 
  - First, start the server:
   - `cd SImpleServer`
   - `yarn start`
 - Then start the React app:
  - run `yarn start` in the base of the project it should open up your default browser to the application
